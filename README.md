# Svelte Dockview Example

An example of using [dockview](https://github.com/mathuo/dockview) with Svelte to create customizable dock layouts with splittable and dockable tabs/windows.

## Features

- Demonstrates the integration of [dockview](https://github.com/mathuo/dockview) with Svelte/SvelteKit (with SSR support).
- Create interactive dock layouts with tab/window management.
